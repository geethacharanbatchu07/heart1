# ❤️ Heart Disease Prediction Web App

🚀 **Live App:** https://heart1-xtfz.onrender.com

---

## 📌 Overview
This project is a Machine Learning-based web application that predicts the risk of heart disease using user-provided health data. The model is trained in Google Colab and deployed using Flask on Render.

---

## ⚙️ Tech Stack
- Python
- Flask
- scikit-learn
- NumPy
- HTML + CSS
- Gunicorn
- Render (Cloud Deployment)

---

## 🧠 Model Details
- Algorithm: Random Forest Classifier
- Trained on heart disease dataset
- Exported using Pickle (`model.pkl`)

### Input Features:
Age, Sex, Chest Pain Type, Blood Pressure, Cholesterol, Fasting Blood Sugar, ECG, Max Heart Rate, Exercise Angina, ST Depression, Slope, Vessels, Thalassemia
